## 4. 为什么最高长期卡在 79.52%（分析）

### 4.1 观察到的事实

- 在两阶段蒸馏体系下，多轮复现实验反复出现“多数配置低于基线，少数组合达到 79.52”。
- 79.52 在至少两条不同蒸馏路径上出现过：
  - a015_lr10（Round-3 修复后）
  - a008_lr10（Round-7 低alpha锚定）
- 说明 79.52 更像“当前数据与流程下的稳态上限”，而不是偶然点。

### 4.2 可能原因

1. 教师信息增益与噪声增益并存
- clean/mismatch 约 577/95（85.86%/14.14%），冲突样本仍会引入不稳定梯度。
- 即便做 selective/anchor，蒸馏收益会被部分噪声抵消。

2. 蒸馏权重与学习率强耦合
- 同一 alpha 在不同 lr 下出现大幅波动，局部最优窄，容易“过蒸馏”或“欠蒸馏”。

3. 训练随机性对上限影响显著
- 不控随机性可偶发 80.72，但方差巨大（std=4.07）。
- 控随机性后方差收敛（std=0.76），但最优回落到基线附近。

4. 学生容量/任务分布限制
- 在当前 7B 架构与现有数据规模下，教师软标签带来的可稳定迁移增益可能接近饱和。

### 4.3 论文推荐表述

- 在稳定性优先口径下，当前可稳定复现的最优成绩为 79.52%。
- 更高单次峰值（80.72%）可达，但复现性不足，不作为主结论。

## 8. 毕业设计正文可直接使用版本

### 8.1 实验设置（正文版）

本研究围绕 DeepSeek 教师指导下的 Qwen2.5-7B-Instruct 学生模型蒸馏展开。任务为牙科单项选择题，输出空间为 A/B/C/D/E。为保证结论的可信性，实验采用“基线对照 + 多轮参数搜索 + 多次复现”的路线，并按阶段记录关键结果。

基线模型为未蒸馏的 Qwen2.5-7B-Instruct，独立测试准确率为 77.11%。教师模型在同任务上的直接测试准确率约为 87.95%。蒸馏阶段采用 LoRA 微调，核心参数包括蒸馏权重 alpha、温度 temperature 与学习率 learning rate。实验过程中先后比较了单阶段选择性蒸馏、两阶段蒸馏（GT 预热 + 选择性蒸馏）、低 alpha 强锚定蒸馏以及纯 SFT 复现策略。

其中，选择性蒸馏规则如下：当教师答案与原始答案一致时使用教师监督；当二者冲突时回退原始答案作为锚点，避免噪声标签直接注入学生模型。统计显示，一致样本约占 85.86%，冲突样本约占 14.14%。

### 8.2 结果分析（正文版）

实验结果表明，蒸馏并非单调提升过程，而是表现出明显的参数敏感性与阶段敏感性。早期单阶段选择性蒸馏仅能追平基线（77.11%），难以形成稳定增益。引入两阶段训练后，模型首次达到 79.52%，相对基线提升 2.41 个百分点，证明“先真值稳定再蒸馏增强”的思路有效。

在后续邻域搜索与复现实验中，79.52% 的结果可再次出现（例如低 alpha 强锚定分支中的 alpha=0.08），但大量组合会回落到 77.11% 以下，说明该任务中的蒸馏收益窗口较窄。进一步地，纯 SFT 分支在未控随机性的条件下曾出现 80.72% 峰值，但同时伴随显著方差（std=4.07），难以作为稳定主结论。为此，本文引入控 seed 与 deterministic 设置，方差收敛到 0.76，但最优仅为 77.11%，表明“高上限”与“高稳定”在当前训练范式下存在显著权衡。

综合多轮实验，本研究在稳定口径下的最优结果为 79.52%。相较 Qwen2.5-7B-Instruct 独立测试 77.11%，绝对提升为 2.41 个百分点，相对提升约 3.12%。

### 8.3 为什么最高长期停留在 79.52%（正文版）

第一，教师增益与教师噪声并存。虽然一致样本占比较高，但冲突样本仍会在训练中引入不稳定梯度，蒸馏信号并非纯净正增益。

第二，超参数耦合显著。alpha 与 learning rate 在本任务中呈强耦合关系，局部最优区域较窄，轻微偏移就可能导致性能明显下滑。

第三，随机性对结果影响较大。未控随机性可偶发高分，但难复现；控随机性后结果稳定，却回到中等水平，说明当前方法尚未同时兼顾上限与鲁棒性。

第四，模型容量与数据分布可能已接近阶段性瓶颈。在现有 7B 学生模型、数据规模和训练轮次设定下，教师软标签可稳定迁移的有效信息可能接近饱和区间。

### 8.4 讨论与局限（正文版）

本研究的主要贡献是完成了从“能跑通蒸馏流程”到“可复现结论”的完整工程闭环，明确了该任务下蒸馏的收益边界与稳定性问题。局限在于：

1. 当前结果仍依赖较强的参数调优，自动化搜索空间有待扩大。
2. 教师标签冲突样本仅采用回退真值策略，尚未引入更细粒度置信度建模。
3. 纯 SFT 高分现象提示存在可挖掘空间，但缺少更系统的方差分解实验。

后续工作可从三个方向展开：

1. 在固定数据与固定 seed 条件下进行更大规模重复试验，报告均值与置信区间；
2. 引入样本级置信度加权蒸馏，进一步抑制冲突样本噪声；
3. 尝试混合损失调度（动态 alpha）以缓解不同训练阶段的目标冲突。

### 8.5 本章小结（正文版）

本章系统评估了 DeepSeek 蒸馏到 Qwen2.5-7B-Instruct 的多阶段方案。实验表明，蒸馏可带来稳定增益上限 79.52%，较基线 77.11% 提升 2.41 个百分点（相对提升 3.12%）。尽管出现过 80.72% 的单次峰值，但其复现性不足，因此本文采用 79.52% 作为主结果，80.72% 作为潜在上限与后续研究依据。
